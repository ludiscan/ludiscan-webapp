# アカウント設定

ludiscan-webappのアカウント設定について説明します。プロフィール情報の管理、セキュリティ設定、APIキーの管理方法を学びましょう。

## アカウント設定へのアクセス

### 設定画面を開く方法

1. **サイドバーから**
   - 画面左側のサイドバーで「プロフィール」または「設定」をクリック

2. **ヘッダーメニューから**
   - 画面右上のユーザーアイコンをクリック
   - 「Profile」を選択

## プロフィール設定

### 基本情報の確認・編集

プロフィールページでは、以下の情報を確認・編集できます：

#### 表示される情報
- **ユーザー名** - アカウント名（Googleアカウント連携時の名前）
- **メールアドレス** - ログインに使用するメールアドレス
- **アカウント作成日** - アカウントが作成された日時
- **最終ログイン** - 最後にログインした日時

#### 編集可能な項目
- **表示名** - アプリケーション内で表示される名前
- **プロフィール画像** - アイコン画像（Googleアカウントの画像が使用されます）
- **タイムゾーン** - 時刻表示に使用するタイムゾーン
- **言語設定** - UIの表示言語（現在は日本語のみ）

### プロフィールの編集方法

1. プロフィールページで「編集」ボタンをクリック
2. 変更したい項目を編集
3. 「保存」をクリック

**💡 ヒント:** 変更は即座に反映されます。

## セキュリティ設定

アカウントのセキュリティを管理します。

### アクセス方法

1. サイドバーから「セキュリティ」を選択
2. または、プロフィールページから「セキュリティ設定」タブをクリック

### セキュリティ機能

#### 1. ログイン履歴

最近のログイン記録を確認できます：

- **ログイン日時** - いつログインしたか
- **IPアドレス** - どこからアクセスしたか
- **デバイス情報** - 使用したブラウザとOS
- **ログイン方法** - Googleアカウント認証

**不審なアクティビティを見つけた場合:**
1. すぐにログアウト
2. Googleアカウントのパスワードを変更
3. 管理者またはサポートチームに連絡

#### 2. アクティブなセッション

現在ログイン中のセッション一覧：

- **現在のセッション** - この端末での現在のセッション
- **他のセッション** - 他のデバイスやブラウザでのセッション

**セッションの管理:**
- 「すべてログアウト」で他の端末からログアウト
- 個別のセッションを選択してログアウト可能

#### 3. 連携アカウント

外部サービスとの連携状況：

- **Google** - ログインに使用（必須）
- **GitHub**（将来的にサポート予定）
- **その他のサービス**（将来的にサポート予定）

### セキュリティのベストプラクティス

#### 推奨設定
- 定期的にログイン履歴を確認
- 使用していないセッションはログアウト
- 公共のPCを使用した後は必ずログアウト

#### 注意事項
- パスワードは他人と共有しない
- 公共Wi-Fiでの使用に注意
- ブラウザに記憶させたパスワードの管理に注意

## APIキー管理

API連携のためのキーを管理します。

### APIキーとは？

APIキーは、外部アプリケーションやスクリプトからludiscan-webappのデータにアクセスするための認証キーです。

### アクセス方法

1. サイドバーから「APIキー」を選択
2. または、プロフィールページから「APIキー」タブをクリック

### APIキーの作成

#### 新しいAPIキーの作成手順

1. APIキー管理ページで「新しいAPIキーを作成」ボタンをクリック
2. APIキー情報を入力：
   - **キー名** - 用途がわかる名前（例：「Python分析スクリプト」）
   - **説明** - 使用目的の詳細（任意）
   - **有効期限** - キーの有効期限（なし/30日/90日/1年）
   - **アクセス権限** - 読み取りのみ/読み書き可能
3. 「作成」をクリック
4. 生成されたAPIキーが表示されます

**⚠️ 重要:** APIキーは作成時に一度だけ表示されます。必ず安全な場所にコピーして保存してください。後で再度表示することはできません。

### APIキーの管理

#### APIキー一覧

作成したAPIキーの一覧が表示されます：

- **キー名** - 作成時に設定した名前
- **作成日** - キーが作成された日時
- **最終使用日** - 最後に使用された日時
- **有効期限** - キーの有効期限
- **ステータス** - 有効/無効/期限切れ

#### APIキーの操作

**有効化/無効化:**
1. APIキー一覧で対象のキーを選択
2. 「無効化」または「有効化」ボタンをクリック
3. 確認ダイアログで確定

**削除:**
1. APIキー一覧で対象のキーを選択
2. 「削除」ボタン（ゴミ箱アイコン）をクリック
3. 確認ダイアログで削除を確定

**⚠️ 注意:** 削除したAPIキーは復元できません。削除すると、そのキーを使用しているアプリケーションやスクリプトは動作しなくなります。

### APIキーの使用方法

#### 基本的な使い方

作成したAPIキーを使って、外部からludiscan-webappのAPIにアクセスできます：

```bash
# 例: curlコマンドでプロジェクト一覧を取得
curl -H "Authorization: Bearer YOUR_API_KEY" \
     ${API_BASE_URL}/v0/project
```

```python
# 例: Pythonでデータを取得
import requests
import os

api_key = "YOUR_API_KEY"
api_base_url = os.environ.get("API_BASE_URL")
headers = {"Authorization": f"Bearer {api_key}"}
response = requests.get(
    f"{api_base_url}/v0/project",
    headers=headers
)
data = response.json()
```

#### APIドキュメント

詳細なAPI仕様については、管理者にお問い合わせください。

### APIキーのセキュリティ

#### セキュリティのベストプラクティス

**保管:**
- APIキーは安全な場所に保管
- コードに直接書かない（環境変数を使用）
- GitHubなどに公開しない

**使用:**
- 必要最小限の権限のみを付与
- 定期的に有効期限付きキーを再作成
- 使用していないキーは削除

**漏洩時の対応:**
1. 直ちにキーを無効化または削除
2. 新しいキーを作成
3. 不正使用がないか確認
4. 必要に応じてサポートチームに連絡

#### 使用状況の監視

APIキーの使用状況を定期的に確認：

1. APIキー管理ページで対象のキーを選択
2. 「使用履歴」を確認
3. 不審なアクセスがないかチェック

表示される情報：
- アクセス日時
- アクセス元IPアドレス
- 実行されたAPIエンドポイント
- レスポンスステータス

## アカウントの削除

アカウントを削除する場合は、以下の手順に従ってください。

### 削除前の確認事項

**⚠️ 重要:** アカウントを削除すると、以下のデータがすべて削除されます：

- プロフィール情報
- プロジェクトへのアクセス権（所有権は管理者に移譲が必要）
- 保存したビュー設定
- エクスポート履歴
- APIキー

### 削除手順

1. セキュリティ設定ページを開く
2. 最下部の「アカウント削除」セクションに移動
3. 「アカウントを削除」ボタンをクリック
4. 確認ダイアログで削除理由を選択（任意）
5. 「削除を確定」をクリック
6. Googleアカウントで再度認証
7. 削除が完了します

**💡 ヒント:** アカウント削除には最大24時間かかる場合があります。この期間内であれば、サポートチームに連絡することでキャンセルできる可能性があります。

## トラブルシューティング

### プロフィール情報が更新できない

**原因:**
- ネットワークエラー
- セッションの期限切れ
- 無効な入力値

**対処法:**
1. ページを再読み込み
2. ログアウトして再ログイン
3. 入力値を確認（特殊文字など）
4. 別のブラウザで試す

### APIキーが作成できない

**原因:**
- 作成可能な上限に達している（最大10個まで）
- 権限不足

**対処法:**
1. 使用していないAPIキーを削除
2. 管理者に権限を確認
3. ページを再読み込み

### APIキーが動作しない

**原因:**
- キーが無効化されている
- 有効期限が切れている
- アクセス権限不足
- APIエンドポイントが間違っている

**対処法:**
1. APIキー管理ページでステータスを確認
2. 有効期限を確認
3. アクセス権限を確認
4. APIドキュメントでエンドポイントを確認
5. 必要に応じて新しいキーを作成

## よくある質問

### Q: Googleアカウントを変更できますか？

A: 現在、ログインに使用するGoogleアカウントの変更はサポートしていません。別のアカウントを使用したい場合は、新しいアカウントを作成してください。

### Q: 複数のアカウントを持てますか？

A: はい。異なるGoogleアカウントで複数のludiscan-webappアカウントを作成できます。

### Q: アカウント情報はどこに保存されますか？

A: アカウント情報はセキュアなサーバー上に暗号化して保存されます。詳細はプライバシーポリシーを参照してください。

### Q: APIキーは何個まで作成できますか？

A: 1つのアカウントにつき、最大10個のAPIキーを作成できます。

### Q: APIキーの有効期限を延長できますか？

A: いいえ。有効期限の延長はできません。有効期限が切れる前に、新しいAPIキーを作成してください。
