## 🎯 機能別ガイド

### データの可視化
1. プロジェクトとセッションを選択
2. ヒートマップビューアーで3D/2Dマップを確認
3. レイヤーとフィルタで表示をカスタマイズ
4. タイムラインで時系列の変化を分析

### データの分析
1. カメラ操作で様々な角度から確認
2. イベントログで詳細なイベント情報を確認
3. フィルタで特定の条件に絞り込み
4. 統計情報で数値的な分析

### データの共有
1. スクリーンショットで視覚的に共有
2. データエクスポートで詳細データを出力
3. ビュー設定を保存して共有
4. レポート作成で包括的な分析結果を提示

## 💡 ヒント

### パフォーマンスの最適化
- 2Dモードは軽量で動作が速い
- 3Dモードは詳細な分析に適している
- フィルタでデータ量を減らすと表示が速くなる
- ブラウザのタブを減らすとパフォーマンスが向上

### 効率的な使い方
- キーボードショートカットを活用（**?** キーで一覧表示）
- ビュープリセットで頻繁に使う視点を保存
- キーフレームで重要な時刻をマーク
- エクスポート機能でデータをバックアップ

### チーム協力
- プロジェクトの共有機能でチームメンバーを招待
- ビュー設定の共有リンクで同じ視点を共有
- エクスポートしたレポートでミーティングに活用
- キーフレームを共有して重要なポイントを伝える
